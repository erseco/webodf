Welcome to WebODF Viewer

WebODF is an AGPL licensed JavaScript library that renders OpenDocument Format (ODF) files using plain HTML and CSS. Browse the sample documents on the left or drop your own .odt file to see the viewer in action.

  • Open source under the GNU AGPL – build custom viewers and editors.
  • Runs entirely in the browser, no plugins or server-side conversion required.
  • Supports desktop and mobile environments.
